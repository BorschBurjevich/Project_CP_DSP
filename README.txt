////////////Описание функциональности и работы ПО///////////
1) Главный проект - CP_CortexA15.
	a)Реализовал работу по таскам. Ядро Cortex A15_0 ожидает получение команды по Ethernet (протокол TCP IP),
	далее идёт обработка команды и выдача результата пользователю.
	б)Управление переферией осуществляется на ядре A15_0.Реализован UART для общение с компасом.
	в)Реализовал тактирование АЦП по интерфейсу SPI.
	г)Реализовал настройку AD9208 по интерфейсу MCSPI.
	д)Реализовал получение данных с АЦП по vip port и дальнейшая обработка данных (спектр, спектр мощности).
	е)Сделал отдельный таск для общения по USB  свнешним SDR приёмником. Общение осуществляется по BULK usb.
	Параллельная работа всех трех тасков требует отладки.
2) Проекты DSP0 и DSP1 выполняют обработку даннных, которые получило главное ядро c АЦП.
	а)В DSP0 происходит расчёт БПФ,наложение окна,операция бабочка.
	б)В DSP1 выполняет такие же операции, также предусмотрен выполнение пеленга для 5 канального пеленгатора.
	в)Ядра всегда ожидают получение команды от ядра Cortex A150. Общение осуществляется с помощью общей памяти.Память выделяется в 
	файле bspCommonBIOS_a15.cfg. Память для ядер DSP0 и DSP1 выделена в файле LORANDIT_CP.bld
